# DBO

# How To Run
1. Clone
2. cd /path/to/project
3. go mod download
4. Adjust DB configuration at .env file
5. go run main.go

# How To Run Using Docker
1. Clone
2. cd /path/to/project
3. Adjust DB configuration at .env file
4. docker-compose up -d
